

to set private key locally, run this command:
export vidly_jwtPrivateKey=mySecureKey

you can generate an auth token by using the post method inside of /routes/auth.js
the response body will be an auth token
you can then copy/paste that auth token into the header of a protected route like so:

inside of headers in request:
HEADER KEY: x-auth-token
HEADER VALUE: [paste token here]

now you can use protected routes!
